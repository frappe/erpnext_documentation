# Qualitätsprüfung


In ERPNext können Sie eingehende und ausgehende Produkte für eine Qualitätsprüfung markieren. Um diese Funktion in ERPNext zu aktivieren, gehen Sie zu:

>  Lagerbestand > Werkzeuge > Qualitätsprüfung > Neu
